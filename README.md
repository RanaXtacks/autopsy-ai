# Autopsy AI

Privacy-first platform that investigates digital behavior patterns from user data.

## Overview

Autopsy AI is designed to help users understand their digital footprint and behavior patterns without compromising their privacy. By processing data locally or in a secure, encrypted environment, it provides insights into how we interact with technology.

## Tech Stack

- **Backend:** Flask (Python)
- **Frontend:** React (JavaScript/TypeScript)
- **Database:** PostgreSQL
- **Analytics:** Pandas
- **Containerization:** Docker

## Getting Started

### Prerequisites

- Docker and Docker Compose
- Python 3.10+
- Node.js 18+

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-org/autopsy-ai.git
   cd autopsy-ai
   ```

2. Start the development environment:
   ```bash
   docker-compose up --build
   ```

3. Access the application:
   - Frontend: `http://localhost:3000`
   - API: `http://localhost:5000`
